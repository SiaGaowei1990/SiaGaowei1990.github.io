必备：github账号一个 域名一个

第一步：
1）首先在github中建立一个 username.github.io的库，只允许建立一个这样的库,其中username就是你github的用户名，也只能是你的username！
新建一个文件 名为CNAME 写入自己的域名地址 如 www.gaoyongwei.site

##这是为了从github向域名做链接##

第二步：配置阿里云域名，因为我购买的是阿里云的.site域名，所以以site举例

2)进入阿里云控制台，进行解析域名，
记录类型 主机记录 解析线路 记录值
CNAME www 默认 siagaowei1990.github.io

##这是为了从域名向github做链接##
